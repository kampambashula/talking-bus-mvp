# Talking Bus Fare Chart MVP

A simple React + Material UI web app that shows official bus fares in Zambia. Includes a QR code passengers or operators can scan to access the live fare chart.

---

## Features

- Responsive fare chart table  
- QR code integration (Python-generated)  
- White, orange, black theme matching Zambian buses  
- About page and footer  
- Ready for Netlify deployment
